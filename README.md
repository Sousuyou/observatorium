# 香の天文台 — Observatorium Botanicum

ジンのボタニカル16種を「音の層」と「軌道」として灯す、純正律の生成アンビエント観測室。
標本札を重ねるほど、和音は壮大になり、渾天儀は賑わう。

**Live:** https://sousuyou.github.io/observatorium/

## 仕組み

- 標本札1枚 = アンビエント1層。全層が5-リミット純正律の同根音上にあり、どの組合せでも調和する
- 鈴が鳴った瞬間、その軌道上の星が閃光する（音と映像の同期）
- FORMULARY（処方箋）: LONDON DRY / JAPANESE CRAFT などジン様式のプリセット
- COMPOUND INDEX: 点灯中の香気成分を集計、成分名クリックで逆引き（TRACE）
- 月齢連動: 実際の月齢を背景に描画。満月の夜は鈴が強くなる
- レシピはURLハッシュ（`#r=juniper.yuzu...`）に同期され、リンク共有で再現できる
- 香気の星座: 成分を共有する標本の主星同士が銀の糸で結ばれる（TRACE中は金）
- 分子の線画: 調書に代表香気成分の骨格構造式を図版として掲載
- 深宇宙の残響: 軌道半径が遠い標本ほど、長い残響の彼方で鳴る
- 律（基準星）: 根音を五つの一等星から選ぶ。点灯中の層はグリッサンドで移調
- ARCHIVE: 実在銘酒の公知ボタニカル構成（タンカレー、ボンベイ・サファイア等）
